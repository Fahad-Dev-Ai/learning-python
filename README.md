# AI Gym Coach Chatbot

A conversational AI chatbot that acts as a professional bodybuilding coach.

## Features
- Multi-turn conversation with memory
- Remembers full chat history
- Type "exit" to quit

## Tech used
- Python
- Groq API
- LLaMA 3 model

## How to run
1. Install groq: pip install groq
2. Add your Groq API key in the code
3. Run: python learning_groq.py
